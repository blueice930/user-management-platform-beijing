大赛申报说明

     

为持久、规范、有效地组织竞赛，“未来科学家大赛”从本届将由“中科国大未来（北京）科技院”承担竞赛具体事宜。为严格赛事，真正评出好的作品，坚持“优化竞赛平台，择优选择作品，顺畅申报渠道，制定监督机制”的原则，以期选出公认的优秀作品。公示期为项目信息纠正期，正式获奖名单发布后不得做任何修改。只有将大赛做得规范、严禁，才能够把这个大赛做成持久的青少年创新教育成果的展示、交流平台。

结合以往申报情况，申报要求有细节变化，申报者一定要按照流程操作，不规范申报的作品将被淘汰。流程如下：

一、  学生把作品上交学校

学生需要提交两个文件：

1、 说明书文件名不需要修改，例如： 太阳能发电公交站 。说明书中不得出现姓名、学校等信息。今年不再需要加出生年月日。

2、学生填写附件二：作品清单，按照案例填写各项内容。

学生核查两个文件，确认无误后将作品和作品清单交给学校负责的教师。教师项目与学生要求相同，不能填写指导教师。

 

二、  学校汇总作品

学校收集学生作品并汇总附件二：作品清单

确定联系人，填写附件三：联系人信息

将作品和对应的作品清单放在一个文件夹下，文件名为：联系人+联系人单位+学段+项数，例: 王红+北京第一实验小学+初中+28。

联系人必须加入大赛qq群（ 231137193 ）
